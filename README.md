# Smart OCR: Оцифровщик студенческих заявлений

Автоматизированная система для распознавания и структурирования университетских заявлений. Скрипт принимает отсканированные бланки (JPEG, PNG, PDF), извлекает рукописный и печатный текст с помощью мультимодальной нейросети Gemini 2.5 Flash и конвертирует данные в чистый, структурированный XML-формат.

## Особенности
* Умное распознавание: Понимает сложный рукописный текст и органично встраивает его в контекст печатного бланка.
* Пакетная обработка: Автоматически сканирует директорию с исходниками и обрабатывает все найденные документы по очереди.
* Структурирование: Самостоятельно разбивает текст на логические блоки (<Header>, <ApplicantInfo>, <Body>, <Signatures>, <Approvals>).
* Изоляция: Работает внутри легковесного Docker-контейнера — не засоряет основную ОС зависимостями.
* Обход блокировок: Поддерживает маршрутизацию трафика через SOCKS5/HTTP прокси.

## Технологии
* Язык: Python 3.12
* Библиотеки: google-genai, PyMuPDF (fitz), httpx[socks]
* Инфраструктура: Docker, GNU Make

## Установка и настройка (Debian / Linux)

### 1. Подготовка сервера
Если на вашем сервере еще не установлен Docker, вы можете развернуть его одной командой:
`make install-docker`

(После выполнения команды потребуется переподключиться к терминалу, чтобы обновились права пользователя).

### 2. Настройка доступа
Откройте файл Makefile и укажите ваши учетные данные в секции run:
1. GEMINI_API_KEY: Вставьте ваш бесплатный ключ от Google AI Studio.
2. https_proxy: Если требуется обход региональных ограничений, актуализируйте IP и порт вашего HTTP/SOCKS5 прокси.

### 3. Подготовка данных
1. Создайте в корне проекта папку БЛАНК.
2. Поместите в неё сканы заявлений (поддерживаются форматы .jpg, .png, .pdf).

## Использование

Для запуска полного цикла (сборка свежего образа и пакетная обработка файлов) выполните:
`make`

### Очистка
Чтобы удалить все сгенерированные XML-файлы и подготовить папку к новому чистому тесту:
`make clean`
